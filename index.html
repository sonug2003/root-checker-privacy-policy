3. How the App Works

Root Checker performs local device checks to determine if your Android device has root access. Here's what happens:

  • Checks for su binary files in common system locations
  • Looks for known root management applications
  • Examines system build properties
  • Tests system partition write access
  • Checks for root-related paths and files

All these checks are performed locally on your device. No information is sent anywhere.

13. Technical Details - How Root Checking Works

Root Checker's core functionality (root detection) is performed entirely locally on your device with no data transmission. The app checks for:

  • Presence of su binary in system paths
  • Known root management app signatures
  • System build tags (test-keys vs release-keys)
  • System partition mount status

Important: These checks do NOT send any information to our servers or third parties. Only the ads and analytics mentioned in this policy involve network communication.
